・Moderator.py
	環境とQ学習に必要なパラメータの設定が行える

・QLearning.py (Q値を学習するフェーズ用)
	

・RobotController.py (学習したQ値を使って行動するフェーズ用)



【実行手順】

・シーンを再生[->]
　　│
　　├ キーボードの[L]ボタンを押す
　　│ Q値を学習するフェーズ
　　│ QLearning.pyが実行される
　　│
　　├ キーボードの[M]ボタンを押す
　　│ 学習したQ値を用いて行動するフェーズ
　　│ RobotController.pyが実行される
　　│

・動作の停止
	キーボードの[Q]ボタンを押す
	ロボットの行動が止まる（再び動くことはない）
