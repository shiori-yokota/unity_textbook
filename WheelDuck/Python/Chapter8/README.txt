・Moderator.py
	迷路の環境とベイズ推定に必要なパラメータの設定が行える
	各状態（マス）の壁の情報の設定を行える
		"上右下左"の順に，壁がある場合は1，壁がない場合は0とする

・BayesianFilter.py
	SIZE は，迷路のサイズである．
	TimeCount は，ロボットが動いた回数である．
	
	
【実行手順】

・シーンを再生[->]
　　│
　　├ キーボードの[S]ボタンを押す
　　│ プログラムが実行される．
　　│ キーボードの矢印ボタンで移動することができる（一度押すと一マス移動）
　　│
　　├ キーボードの[Q]ボタンを押す
　　│ ロボットの行動が止まる（再び移動させることはできません）
　　│
